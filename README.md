## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY_1` and `GEMINI_API_KEY_2` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
